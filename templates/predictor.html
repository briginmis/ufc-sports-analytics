Fight predictor

Model overview


The fight predictor uses a linear regression model and is 63% accurate.
The model was trained using fight data from 2010 to 2021.
Prediction will be based on the latest available data as at 2021.
The model does not consider weight class in its prediction.

How to use this model


1. Select two fighters from the drop down menus.
2. Click "Predict Winner" button.
